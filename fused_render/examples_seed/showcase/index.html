Example projects

Working projects you can open and pick apart — each one is a page plus a Python file in a folder, like everything in Fused Render.

Clicking a card fetches that example from GitHub into ~/Documents/Fused and opens it. A confirm screen appears first.

Geospatial

Maps, hexagons, rasters, and public datasets — analysis that runs on your machine.

Local tools

Fused Render pointed at your own machine — files, processes, and settings as web pages.

Browse the source of every example at github.com/fusedio/fused-render-examples.
