Fused Render

A local file explorer with Python-powered interactive HTML views — browse any directory in the browser, preview your data, and author pages that call local Python.

macOS

Download for macOS

Signed and notarized app, macOS 11+. Bundles its own Python, the fused CLI, and rclone — nothing else to install.

Or with Homebrew:

brew install --cask fusedio/tap/fused-render

Python package (macOS, Windows, Linux)

Requires Python 3.10+. Install the wheel, then run fused-render:

pip install <wheel-url>  # copy the wheel URL from the latest release

On Windows, fused-render-open --register adds Explorer "Open with" integration for every format it previews.
